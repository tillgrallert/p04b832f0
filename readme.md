This repository holds supplementary material such as maps, data sets, and code for a paper tentatively titled "Urban food riots in late Ottoman *Bil훮d al-Sh훮m* as a 'repertoire of contention'". All materials are licensed as [cc by-nd 4.0](http://creativecommons.org/licenses/by-nd/4.0/).

Material is organised in the following folders

- `data/`: data sets on food prices as CSV.
- `maps/`: maps/geo-located data for the cities of Damascus, Hama, and Homs - mostly as GeoJSON.
- `plots/`: plots of time series of various food prices across *Bil훮d al-Sh훮m*.
- `r/`: R scripts for plotting the price data on a timeline.
